# agileengine_test

Requirements:
Python 3.8+

1) Create `virtualenv` (https://virtualenv.pypa.io/en/latest/installation.html)
2) Activate virtualenv (`source env/bin/activate`)
3) Install project dependencies: 

`pip install -r requirements.txt `

4) Run app:

`./run.sh`

5) Run Celery:

`./run_celery.sh`
